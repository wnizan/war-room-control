=================================================================
                      METRIC DEFINITIONS
=================================================================

FPS (Frames Per Second)
  Calculation:              Count frames, reset every 1 second
  Unit:                     fps
  Update frequency:         1 per second
  Color thresholds:         Green >=50, Yellow 30-49, Red <30

FRAME TIME
  Calculation:              now - lastFrameTime (milliseconds)
  Unit:                     ms
  Update frequency:         Every frame
  Color thresholds:         Green <=20ms, Yellow 20-33ms, Red >33ms

JS HEAP SIZE
  Calculation:              performance.memory.usedJSHeapSize / 1,048,576
  Unit:                     MB
  Update frequency:         Every frame
  Browser support:          Chrome only (shows "N/A" otherwise)

UPDATE RATE
  Calculation:              Count TickUpdate messages, reset every 1 second
  Unit:                     Hz (messages/second)
  Update frequency:         1 per second
  Color thresholds:         Green >=30, Yellow 15-29, Red <15

LONG TASKS
  Calculation:              Count PerformanceObserver entries > 50ms
  Unit:                     count
  Update frequency:         Event-driven
  Display:                  Only shown if count > 0

=================================================================
                   FALLBACK RULES & ERROR HANDLING
=================================================================

1. performance.memory (Chrome-only)
   Fallback:                Show "N/A"
   Graceful:                Yes (silent)

2. PerformanceObserver
   Fallback:                Skip initialization
   Graceful:                Yes (try-catch)

3. longtask entryType
   Fallback:                Catch exception, continue
   Graceful:                Yes (try-catch)

4. requestAnimationFrame
   Fallback:                None (available in all modern browsers)
